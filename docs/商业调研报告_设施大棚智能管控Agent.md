## 一、项目概述

### 1.1 项目背景

设施农业（温室大棚）是现代农业的重要组成部分，但目前大棚环境管控（温湿度、光照、通风、水肥）主要依赖人工经验，存在以下痛点：

- **控制参数多且相互耦合**：温度、湿度、光照、C02浓度、土壤EC值、pH值等参数相互影响，人工难以全面把握
- **响应滞后**：24小时需要专人看管，突发天气变化时难以及时调整
- **水肥浪费严重**：传统灌溉过量或不足，导致水肥浪费和产量损失
- **农产品品质不稳定**：环境控制不一致导致农产品品质参差

### 1.2 解决方案

开发设施大棚智能管控 Agent，基于多源数据（环境传感器、天气预报、作物模型、历史数据）自动生成控制建议或直接控制设备（风机、遮阳帘、补光灯、滴灌系统）。

**适用作物**：蔬菜、草莓、番茄、菌菇、花卉
